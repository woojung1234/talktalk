TalkTalk 개인정보처리방침

최종 업데이트: 2025년 6월 2일

요약: TalkTalk은 사용자의 개인정보를 수집하지 않으며, 모든 대화 생성은 익명으로 처리됩니다.

1. 개인정보 수집 및 이용

1.1 수집하는 개인정보

TalkTalk 앱은 다음과 같은 개인정보를 수집하지 않습니다:

  • 이름, 전화번호, 이메일 주소 등 개인 식별 정보
  • 위치 정보
  • 연락처 정보
  • 기기 고유 식별자

1.2 자동 수집 정보

앱 개선을 위해 다음 정보만 익명으로 수집할 수 있습니다:

  • 앱 사용 통계 (익명)
  • 오류 로그 (개인정보 제외)
  • 기기 유형 및 OS 버전 (통계 목적)

2. 대화 내용 처리

2.1 AI 서비스 이용

대화 주제 생성을 위해 OpenAI GPT API를 사용합니다:

  • 사용자가 입력한 상황 정보는 OpenAI 서버로 전송됩니다
  • 전송되는 정보에는 개인 식별 정보가 포함되지 않습니다
  • OpenAI의 개인정보처리방침이 적용됩니다
  • 대화 내역은 앱에 저장되지 않습니다

2.2 데이터 보관

TalkTalk은 다음과 같이 데이터를 처리합니다:

  • 생성된 대화 주제는 기기에만 임시 저장됩니다
  • 앱을 종료하면 모든 데이터가 삭제됩니다
  • 서버에 사용자 데이터를 저장하지 않습니다

3. 개인정보 제3자 제공

TalkTalk은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외입니다:

  • OpenAI API 서비스 이용을 위한 익명 데이터 전송
  • 법적 의무 이행을 위해 필요한 경우

4. 개인정보 보호

4.1 보안 조치

  • 모든 API 통신은 HTTPS로 암호화됩니다
  • 개인 식별 정보는 수집하지 않습니다
  • 앱 내 데이터는 기기에서만 처리됩니다

4.2 데이터 보관 기간

개인정보를 수집하지 않으므로 별도의 보관 기간이 없습니다. 앱 사용 중 생성되는 임시 데이터는 앱 종료 시 자동 삭제됩니다.

5. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 앱 삭제를 통한 모든 데이터 완전 제거
  • 개인정보처리방침에 대한 문의
  • 앱 사용 중단

6. 아동의 개인정보 보호

TalkTalk은 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 만약 13세 미만 아동의 개인정보가 수집된 것을 발견하면 즉시 삭제 조치를 취합니다.

7. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있습니다. 중요한 변경사항이 있을 경우 앱 내 공지를 통해 알려드립니다.
